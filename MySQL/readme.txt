1. Install and configure your IDE to use the Anaconda (version 3) Python interpreter. Dependencies can be installed separately, but Anaconda is recommended for ease of use.

2. Open data_generator.py

Insert your own database credentials in lines 13-16
	
3. Open run_script.py

s and n, on lines 16 and 17 respectively, are the 'starting index' and the total number of new rows. Make sure the starting index is a value higher than the highest indexed row in LOG_DETAILS_T

Edit the arguments from lines 31-42 as necessary. A list of arguments and their significance can be found in the comments at the top of the script. It is recommended that these arguments remain similar to what's been given.

4. Run run_script.py
